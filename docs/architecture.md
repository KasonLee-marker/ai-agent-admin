## 模块说明

### 1. Prompt 管理模块
- Prompt 模板 CRUD
- 版本控制（历史版本回溯）
- 分类标签管理
- 变量占位符支持

### 2. 模型管理模块
- 多模型配置（百炼、OpenAI、DeepSeek 等）
- 动态参数调整（temperature、max_tokens 等）
- API Key 管理
- 模型健康检查

### 3. 对话调试模块
- 多轮对话会话
- 流式响应 (SSE)
- 对话历史记录
- Prompt 效果对比

### 4. 数据集管理模块
- 数据导入（JSON、CSV、Excel）
- 数据版本控制
- 数据项 CRUD
- 数据集导出

### 5. 评估模块
- 评估器配置（准确率、延迟、token 消耗）
- 批量实验执行
- 结果统计和可视化
- A/B 测试支持

### 6. 可观测性模块（简化）
- 调用日志记录
- 链路追踪（OpenTelemetry Lite）
- 性能指标监控

## API 设计原则

1. RESTful 风格
2. 统一响应格式
3. 分页查询支持
4. 错误码规范

## 安全设计

1. 内网部署，简化认证
2. API Key 加密存储
3. 请求限流保护
4. 操作日志审计
