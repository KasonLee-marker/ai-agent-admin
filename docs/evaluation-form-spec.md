# 评估任务表单规格文档

## 1. 概述

评估任务表单用于创建和编辑模型评估任务，支持基础评估和RAG评估两种模式。

## 2. 字段定义

### 2.1 基础字段

| 字段名              | 类型     | 必填   | 默认值      | 说明                   |
|------------------|--------|------|----------|----------------------|
| name             | String | ✅ 必填 | -        | 任务名称，最大200字符         |
| description      | String | ❌ 可选 | -        | 任务描述，最大1000字符        |
| datasetId        | String | ✅ 必填 | -        | 数据集ID，提供测试数据         |
| promptTemplateId | String | ❌ 可选 | null     | Prompt模板ID，不选则使用默认模板 |
| modelConfigId    | String | ❌ 可选 | 系统默认对话模型 | 对话模型ID，用于生成回答        |

### 2.2 RAG评估字段

| 字段名              | 类型      | 必填    | 默认值             | 说明                      |
|------------------|---------|-------|-----------------|-------------------------|
| knowledgeBaseId  | String  | 条件必填* | null            | 知识库ID，用于检索上下文           |
| enableRag        | Boolean | ❌ 可选  | false           | 是否启用RAG评估模式             |
| embeddingModelId | String  | ❌ 可选  | 系统默认Embedding模型 | Embedding模型ID，用于语义相似度计算 |

*注：knowledgeBaseId 在 enableRag=true 时变为必填

## 3. 字段联动规则

### 3.1 enableRag 开关联动

| 条件                | 联动行为                                  |
|-------------------|---------------------------------------|
| enableRag = true  | knowledgeBaseId 字段变为必填，显示红色提示"请选择知识库" |
| enableRag = true  | knowledgeBaseId 下拉框高亮显示（边框变蓝）         |
| enableRag = true  | 显示信息提示："RAG模式需要选择知识库进行文档检索"           |
| enableRag = false | knowledgeBaseId 字段恢复为可选，清空之前的必填提示     |
| enableRag = false | knowledgeBaseId 选择值保留但不再强制要求          |

### 3.2 数据集选择联动

| 条件                                | 联动行为                             |
|-----------------------------------|----------------------------------|
| 数据集有期望输出(itemCount > 0且有output字段) | 显示提示："建议选择Embedding模型以计算语义相似度"   |
| 数据集无期望输出                          | 显示提示："当前数据集无期望输出，无法计算语义相似度和AI评分" |

### 3.3 Embedding模型推荐

| 条件                                      | 联动行为                               |
|-----------------------------------------|------------------------------------|
| enableRag = true 且 embeddingModelId 未选择 | 显示提示："RAG评估建议选择Embedding模型计算语义相似度" |
| 数据集有期望输出 且 embeddingModelId 未选择         | 显示提示："建议选择Embedding模型"             |

### 3.4 知识库选择联动

| 条件                                      | 联动行为                      |
|-----------------------------------------|---------------------------|
| knowledgeBaseId 已选择 且 enableRag = false | 显示提示："已选择知识库，建议开启RAG评估模式" |
| knowledgeBaseId 已选择 且 enableRag = true  | 无特殊提示，正常状态                |

## 4. 表单验证规则

### 4.1 提交时验证

```javascript
function validateForm(values) {
  const errors = [];
  
  // 必填字段验证
  if (!values.name?.trim()) {
    errors.push("名称不能为空");
  }
  if (!values.datasetId) {
    errors.push("请选择数据集");
  }
  
  // RAG模式联动验证
  if (values.enableRag && !values.knowledgeBaseId) {
    errors.push("启用RAG评估时必须选择知识库");
  }
  
  return errors;
}
```

### 4.2 实时验证（字段变化时）

| 字段变化                                 | 验证行为                                |
|--------------------------------------|-------------------------------------|
| enableRag 变为 true                    | 立即检查 knowledgeBaseId 是否已选择，未选择则显示警告 |
| knowledgeBaseId 变为空 且 enableRag=true | 立即显示"请选择知识库"错误提示                    |

## 5. UI交互设计

### 5.1 表单布局

```
┌─────────────────────────────────────────────────────┐
│ [名称]              (必填，文本输入框)                │
├─────────────────────────────────────────────────────┤
│ [描述]              (可选，多行文本框)                │
├─────────────────────────────────────────────────────┤
│ [数据集]            (必填，下拉选择)                  │
│  └─ 选择后显示：数据项数、是否有期望输出              │
├─────────────────────────────────────────────────────┤
│ [Prompt模板]        (可选，下拉选择，可清空)          │
├─────────────────────────────────────────────────────┤
│ [对话模型]          (可选，下拉选择，默认使用系统默认) │
├─────────────────────────────────────────────────────┤
│ ─────────── RAG评估配置 ───────────                 │
├─────────────────────────────────────────────────────┤
│ [知识库]            (条件必填*，下拉选择)             │
│  └─ enableRag=true时显示必填标记                     │
├─────────────────────────────────────────────────────┤
│ [Embedding模型]     (可选，下拉选择)                  │
│  └─ 根据条件显示推荐提示                              │
├─────────────────────────────────────────────────────┤
│ [启用RAG评估] ○──── (开关控件)                       │
│  └─ 开启后触发联动验证                               │
└─────────────────────────────────────────────────────┘
```

### 5.2 提示信息样式

| 提示类型 | 样式             | 示例                 |
|------|----------------|--------------------|
| 信息提示 | 蓝色文字，info图标    | "建议选择Embedding模型"  |
| 警告提示 | 橙色文字，warning图标 | "已选择知识库，建议开启RAG评估" |
| 错误提示 | 红色文字，error图标   | "启用RAG评估时必须选择知识库"  |

## 6. 评估流程说明

### 6.1 基础评估流程（enableRag=false）

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  数据集输入  │ →  │  Prompt渲染  │ →  │  对话模型    │
│  (问题)     │     │  (模板+问题) │     │  (生成回答)  │
└─────────────┘     └─────────────┘     └─────────────┘
                                               ↓
                    ┌─────────────────────────────────────┐
                    │  计算指标:                           │
                    │  - 语义相似度 (需期望输出+Embedding) │
                    │  - AI评分 (需期望输出)               │
                    └─────────────────────────────────────┘
```

### 6.2 RAG评估流程（enableRag=true）

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  数据集输入  │ →  │  知识库检索  │ →  │  RAG提示词   │
│  (问题)     │     │  (Top-K文档) │     │  (上下文+问题)│
└─────────────┘     └─────────────┘     └─────────────┘
                                               ↓
                    ┌─────────────┐     ┌─────────────┐
                    │  对话模型    │ →  │  生成回答    │
                    └─────────────┘     └─────────────┘
                                               ↓
                    ┌─────────────────────────────────────┐
                    │  计算指标:                           │
                    │  - 检索得分 (需期望文档ID)           │
                    │  - 事实忠实度                        │
                    │  - 语义相似度 (需期望输出+Embedding) │
                    │  - AI评分 (需期望输出)               │
                    └─────────────────────────────────────┘
```

## 7. 默认值策略

| 字段               | 默认值来源                                    | 优先级         |
|------------------|------------------------------------------|-------------|
| modelConfigId    | 系统默认对话模型(isDefault=true)                 | 用户选择 > 系统默认 |
| embeddingModelId | 系统默认Embedding模型(isDefaultEmbedding=true) | 用户选择 > 系统默认 |
| promptTemplateId | 无默认，使用内置默认模板                             | 用户选择 > 内置模板 |
| enableRag        | false                                    | 用户选择        |

## 8. 编辑模式特殊处理

| 场景                                    | 处理方式                  |
|---------------------------------------|-----------------------|
| 编辑已有任务                                | 加载所有已保存字段值            |
| enableRag=true 但 knowledgeBaseId=null | 显示警告，提示数据不一致          |
| 任务状态为RUNNING                          | 禁止编辑，显示提示"运行中的任务无法编辑" |
| 任务状态为COMPLETED                        | 允许编辑，但提示"修改后需重新运行评估"  |
