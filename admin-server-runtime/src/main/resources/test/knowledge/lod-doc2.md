### 比尔吉沃特（外港/黎明）

"独眼鲨"在外港停下，陆星野和艾克换乘一艘小艇，划向比尔吉沃特的码头。

随着距离拉近，比尔吉沃特的全貌展现在他们眼前。

那是一座建立在巨大礁石上的城市，建筑层层叠叠，从海平面一直延伸到云端。破旧的木屋和华丽的石堡交错在一起，吊桥和绳索连接着各个区域。码头上停满了各种船只，从简陋的渔船到巨大的战舰，应有尽有。
